# 개발 로드맵

## 비전

UI 에셋 워크플로를 자동화하고 시간이 지날수록 기능 범위를 확장하는 범용 에디터 전용 Unreal MCP 플러그인을 구축합니다.

## 원칙

- 에디터 전용 범위(런타임 모듈 없음)
- 드라이런과 명확한 변경 리포트를 갖춘 안전한 기본 동작
- 기본적으로 사용자가 수정한 위젯 에셋 보존
- 강한 테스트 커버리지를 갖춘 점진적 전달

## 단계

### 1단계 - 패치 우선 MVP

- 폴더 경로 기반 이미지 임포트 규칙 구현(프로젝트 레벨, 다중 대상)
- 임포트/재임포트 시 UI 텍스처 임포트 설정 자동화
- C++ `UUserWidget` 클래스에서 Widget Blueprint 생성
- 위젯 트리 및 바인딩 변경은 `patch` 모드에서만 적용
- MCP 관리 노드에 한해 복구/리페어 및 리패런트 작업 추가

### 2단계 - 안정성 및 툴링

- 멱등성 검사 및 충돌 리포팅 추가
- 검증 및 드라이런 diff 출력 확장
- 임포트, 위젯 생성, 바인딩, 복구 경로에 대한 회귀 테스트 추가
- 진단 정보 및 명령 단위 에러 코드 개선

### 3단계 - 범위 확장

- UI 자동화 기능 확장(스타일 규칙, 에셋 네이밍, 구조 유틸리티)
- 기존 프로젝트를 위한 고급 마이그레이션/복구 워크플로 추가
- 배치 작업 및 팀 단위 사용성 개선
