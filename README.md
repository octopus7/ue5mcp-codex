# ue5mcp-codex

Minimal Unreal Engine 5.7 project shell with the base game module and `BasicMap`.

## Project State

- Startup map: `MCPDemoProject/Content/Maps/BasicMap.umap`
- Editor startup map: `BasicMap`
- Default game mode: `GameModeBase`

All MCP tooling, generated UI assets, and demo UI runtime code have been removed.
